# 2 x n 타일링

2×n 크기의 직사각형을 1×2, 2×1 타일로 채우는 방법의 수를 구하는 프로그램을 작성하시오.

## 생각해보자

1. n=1 일때

2행 1열짜리는 채우는 방법은 1\*2 한개

dp[1] = 1

2. n=2 일때

2행 2열짜리는 채우는 방법은 2

dp[2] = 2

3. n=3 일때

2행 3열짜리를 채우는 방법은 dp[2] 짜리로 2행 2열을 채우고 dp[1]ㅉ리로 2행 1열을 채우기

dp[3] = dp[2] + dp [1]

4. 그러면 n =?

dp[n] = dp[n-1] + dp[n-2]
